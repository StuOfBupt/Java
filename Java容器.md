##### 2、HashMap和Hashtable

- HashMap
  - 非线程安全
  - 键和值允许null存在
  - 效率高
- Hashtable
  - 线程安全
  - 键和值不允许null
  - 效率低

##### 3、ArrayList和Vector

- ArrayList
  - 非线程安全
  - 效率高
- Vector
  - 非线程安全
  - 效率低
